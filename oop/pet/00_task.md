# Задача: 

Управление домашними питомцами

1) В системе есть один домашний питомец
2) Питомец может быть разных видов : Собака или Кошка.
3) Каждый требует индивидуального ухода и имеет особенности питания
4) Нужно создать систему, которая принимает любого питомца и позволяет взаимодействовать с ним с учетом его особенностей.
