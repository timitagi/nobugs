# Принципы ООП

Инкапсуляция: детали уход(), кормление(), производить() скрыты внутри конкретных животных
Наследование: Корова и Курица наследуют Домашнее животное
Полиморфизм: вызов уход(), кормление(), производить() - дают разный результат для разного вида животного
Абстракция: Ферма взаимодействует с животным через контракт Домашнее животное
