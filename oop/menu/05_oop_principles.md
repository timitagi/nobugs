# Принципы ООП

Инкапсуляция: формирование описания температуры и объема скрыты внутри конкретных блюд
Наследование: Горячее блюдо и Напиток наследуют(расширяют) Блюдо.
Полиморфизм: вызов описание() дает разный результат для разных типов блюд.
Абстракция: Меню взаимодействует с блюдом через Блюдо.описание().
